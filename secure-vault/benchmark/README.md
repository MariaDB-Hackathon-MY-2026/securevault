# SecureVault Benchmark Package

This directory contains evaluator-facing benchmark material for the SecureVault project.

## Included Reports

- `semantic-search-benchmark-latest.md`
  Retrieval latency benchmark after indexing has already completed.
- `semantic-pipeline-accuracy-latest.md`
  End-to-end semantic indexing and retrieval benchmark using the live Google embedding provider.

## Included Guides

- `semantic-search-benchmark-guide.md`
  Methodology and interpretation notes for the retrieval latency benchmark.
- `semantic-pipeline-accuracy-guide.md`
  Methodology and interpretation notes for the full semantic pipeline benchmark.

## Scope

The retrieval benchmark documents responsiveness after indexing.

The semantic pipeline benchmark documents indexing behavior and retrieval quality.

Together, the two reports separate system responsiveness from end-to-end semantic ranking quality.
